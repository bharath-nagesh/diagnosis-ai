This directory should contain app icons for PWA support in various sizes.

- icon-72x72.png

- icon-128x128.png
- icon-144x144.png
- icon-152x152.png

## Creating Icons

You can create these icons using design tools like:
- Figma
- Adobe Illustrator
- Sketch
- Canva

Or use online tools to generate all sizes from a single high-resolution image.

## Icon Design Guidelines

- Use the app's gradient colors: #667eea to #764ba2
- Include a recognizable medical/diagnostic symbol
- Ensure icons are clear and visible at all sizes
- Follow iOS and Android icon guidelines
